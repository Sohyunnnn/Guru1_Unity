
# 몽환: 만개의 복
- 몽환: 만개의 복은 3D 게임으로, 코마 상태에 빠진 주인공이 생과 사 사이의 섬에서 깨어나 그 섬의 주민인 여우들의 도움을받아 부서진 배를 고쳐 섬을 빠져나오는 스토리의 게임입니다.
